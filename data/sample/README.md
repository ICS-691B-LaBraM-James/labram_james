# Sample EEG Data

Place sample EEG files for testing in this directory.

## Supported Formats

- `.edf` — European Data Format
- `.fif` — MNE-Python native format
- `.set` — EEGLAB format
- `.csv` — Channel x time CSV exports

## Where to Find Free Sample EEG Files

- [PhysioNet](https://physionet.org) — large collection of freely available physiological signal datasets including EEG
- [OpenNeuro](https://openneuro.org) — BIDS-formatted neuroimaging datasets
- [TUH EEG Corpus](https://isip.piconepress.com/projects/tuh_eeg/) — Temple University Hospital EEG dataset (requires registration)
